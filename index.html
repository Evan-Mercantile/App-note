⚙️ Configuration Firebase requise

Pour activer la sync multi-appareils et les comptes, connecte l'app à un projet Firebase gratuit.


Étapes (5 min) :


1. Va sur console.firebase.google.com

2. Créer un projet → donne-lui un nom

3. Dans Authentication → Commencer → active Email/Mot de passe

4. Dans Firestore Database → Créer une base → mode Production

5. Dans Règles Firestore, colle ceci :

rules_version = '2'; service cloud.firestore { match /databases/{db}/documents { match /users/{uid}/{doc=**} { allow read, write: if request.auth.uid == uid; } } }

6. Dans Paramètres du projet → Ajouter une app → Web → copie la config

7. Ouvre ce fichier HTML dans un éditeur, trouve COLLE_TA_CONFIG_ICI et remplace la config.
